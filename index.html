AI + LLM Engineering

Mixture of Experts Transformer (MoE)

Built a sparse Mixture of Experts (MoE) Transformer from scratch in PyTorch, implementing Top-K routing with noisy gating for load balancing. Designed token-level expert dispatch, enabling sparse computation where only selected experts process each token. Integrated MoE layers into a GPT-style architecture with causal self-attention, improving scalability and efficiency similar to modern large-scale LLMs.

AI + Deep Learning

Transformer Attention + KV Cache Optimization

Built Multi-Head Attention from scratch in PyTorch and implemented KV cache optimization for efficient autoregressive inference. Benchmarked both approaches, achieving ~2.7× speedup by eliminating redundant key/value computations. Explores core transformer internals used in modern LLMs.
